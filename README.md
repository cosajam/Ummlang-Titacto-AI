# Ummlang-Titacto-AI
엄랭으로 틱택토 AI는 아니고 틱택토 사이버 노가다 꾼을 만들어 보았습니다.

###사용법
main.umm의 121번 라인에 case.umm 전체 내용을 삽입하고 
https://www.ryugod.com/pages/ide/umm에서 실행시키세요
초반 코드의

엄
어엄
어어엄
........

부분에 **엄 = 첫째 칸   어엄 = 둘째 칸** 으로 치고 **빈칸 = "_"  X = "."  O = ".."** 를
변수에 할당해 인풋 대용으로 사용해 실행버튼 누르시면 현재 상황 출력과 다음 최선의 수를 계산해 줍니다.
